House Price Prediction

Linear Regression Model This project demonstrates the use of a Linear Regression model to predict a continuous target variable based on one or more predictor variables. The model leverages the scikit-learn library for training and evaluating the performance of the regression algorithm.
Approach: Data Preprocessing: The data was cleaned and prepared for model training by handling missing values, scaling features, and encoding categorical variables.
Model Training: The linear regression model was trained on the dataset using the Ordinary Least Squares (OLS) method, which minimizes the sum of squared differences between the predicted and actual values.
Evaluation: Model performance was evaluated using common metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and R-squared (R²) score to assess how well the model can predict unseen data.
Technologies Used: Python: Main programming language
Libraries: scikit-learn, Pandas, NumPy, Matplotlib

Classification Model

Titanic Survival Prediction (Logistic Regression Model)
This project demonstrates the use of a Classification model to predict whether a passenger survived or not on the Titanic, based on various features such as age, sex, class, and other passenger attributes. The model leverages the scikit-learn library for building and evaluating the classification algorithm.
Objective:
The goal of this project is to predict the survival of passengers on the Titanic based on the dataset available from the Kaggle Titanic competition. The challenge focuses on classifying passengers into two classes: Survived (1) or Did Not Survive (0).
Approach:
Data Preprocessing: The Titanic dataset was cleaned and preprocessed to handle missing values, encode categorical variables (e.g., "Sex," "Embarked"), and scale numerical features. The training and testing datasets were split for validation purposes..
Model Training:
The model was trained on the features of the passengers, such as Age, Sex, Pclass, SibSp, Fare, and others.
Evaluation:
Model performance was evaluated using metrics such as Accuracy, Precision Score.
